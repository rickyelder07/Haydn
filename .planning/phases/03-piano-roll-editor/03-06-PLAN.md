  <how-to-verify>

2. Upload a MIDI file (use a test file with multiple tracks)

**Visual Rendering:**
3. Verify piano roll appears below transport controls
4. Check that notes display as colored rectangles at correct pitch/time positions
5. Check that piano keys sidebar shows note names (C labels visible)
6. Check grid lines visible (bar lines bold, beat lines medium)

**Track Selection:**
7. Click different tracks in the Track List section
8. Verify piano roll updates to show the selected track's notes
9. Verify selected track has visual highlight (blue ring)

**Adding Notes:**
10. Click on empty space in the piano roll
11. Verify a new note appears at the clicked pitch and time
12. Check the note snaps to the grid

**Deleting Notes:**
13. Right-click on an existing note
14. Verify the note is removed
15. Try shift+click on a note - should also delete

**Moving Notes:**
16. Click and drag a note to a new position
17. Verify the note moves to the new pitch and/or time
18. Check that the move snaps to grid

**Undo/Redo:**
19. After adding a note, press Ctrl+Z (or Cmd+Z on Mac)
20. Verify the added note disappears
21. Press Ctrl+Shift+Z (or Cmd+Shift+Z)
22. Verify the note reappears
23. Test undo/redo toolbar buttons work too

**Zoom:**
24. Use zoom controls to zoom in horizontally
25. Verify grid shows finer subdivisions when zoomed in
26. Use zoom controls to zoom out
27. Verify grid shows only bar lines when zoomed out
28. Test vertical zoom - note height should change

**Note Inspector:**
29. Click a note to select it
30. Check the inspector panel shows note properties (pitch, position, duration, velocity)
31. Change duration value - verify note width updates
32. Change velocity slider - verify note color changes

**Playback Integration:**
33. Press play - verify playhead line moves across the piano roll
34. Notes should still be visible during playback

**Keyboard Shortcuts:**
35. Select a note, press Delete/Backspace - note should be removed
36. Verify Space still toggles play/pause
