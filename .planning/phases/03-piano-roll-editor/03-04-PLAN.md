**ZoomControls.tsx:**

Props:
- `zoomX: number` - Current horizontal zoom (0.25 to 4.0)
- `zoomY: number` - Current vertical zoom (0.5 to 3.0)
- `onZoomXChange: (zoom: number) => void`
- `onZoomYChange: (zoom: number) => void`

Render a compact control bar with:
- Horizontal zoom: [-] button, display "H: {zoomX.toFixed(1)}x", [+] button
- Vertical zoom: [-] button, display "V: {zoomY.toFixed(1)}x", [+] button
- [Reset] button that sets both to 1.0
- Zoom steps: multiply/divide by 1.25 on each click
- Clamp zoomX to [0.25, 4.0], zoomY to [0.5, 3.0]

Style: small buttons with Tailwind (px-2 py-1 text-sm), inline layout, subtle background.

**NoteInspector.tsx:**

Props:
- `note: HaydnNote | null` - The currently selected note (null = no selection)
- `noteIndex: number | null`
- `ppq: number` - For duration display conversion
- `onUpdate: (noteIndex: number, updates: Partial<HaydnNote>) => void`
- `onDelete: (noteIndex: number) => void`

Display when note is selected:
- **Pitch**: Read-only display of note name (use midiToNoteName from timeConversion.ts) + MIDI number. Example: "C4 (60)"
- **Position**: Read-only display of tick position + bar:beat:sixteenth (use ticksToBarsBeat)
- **Duration**: Editable number input showing duration in ticks. Also show human-readable: "{durationTicks/ppq} beats". On change, call onUpdate with new durationTicks.
- **Velocity**: Editable range slider (0.0 to 1.0, step 0.05). Display percentage. On change, call onUpdate with new velocity.
- **Delete** button: calls onDelete

When no note selected, show: "Select a note to view properties"

Style: compact panel with labels and inputs. Use Tailwind form styling. Small text (text-xs/text-sm). Light background card.

Both files need 'use client' directive.
