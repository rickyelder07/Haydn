**src/components/SynthDebugPanel/SynthDebugPanel.tsx:**

'use client' component. Uses React useState for slider values. Calls getSynthInstruments() from NoteScheduler to get live instrument references.

**Structure:**
- Floating panel, fixed position bottom-right (bottom-4 right-4), z-50
- Background: `bg-[rgba(17,24,39,0.97)]` (matches FloatingEditPanel pattern), `border border-white/10 rounded-xl`
- Width: ~360px, max height: 80vh with overflow-y-auto
- Header: "Synth Debug" label + close button (X)
- Collapsible: clicking header toggles open/collapsed state

**Per-category sections.** Each category has:
- Category name header (e.g., "Piano", "Strings", "Brass")
- ADSR sliders: attack (ms), decay (ms), sustain (0-1), release (ms)
- Filter cutoff slider (Hz)
- For Strings: vibrato depth slider (0-0.3 normalRange)

**Categories to include:**
```
piano: { programRange: '0-7', cutoff: 3800, adsr: { attack: 0.005, decay: 0.25, sustain: 0.3, release: 1.2 } }
strings: { programRange: '40-55', cutoff: 2500, adsr: { attack: 0.25, decay: 0.1, sustain: 0.85, release: 1.5 }, vibrato: 0.1 }
brass: { programRange: '56-79', cutoff: 3000, adsr: { attack: 0.04, decay: 0.15, sustain: 0.65, release: 0.5 } }
organ: { programRange: '16-20', cutoff: 3000, adsr: { attack: 0.01, decay: 0, sustain: 1.0, release: 0.2 } }
guitar: { programRange: '24-31', cutoff: 3500, adsr: { attack: 0.008, decay: 0.3, sustain: 0.4, release: 0.8 } }
bass: { programRange: '32-39', cutoff: 2800, adsr: { attack: 0.015, decay: 0.1, sustain: 0.75, release: 0.6 } }
```

**Slider ranges:**
- attack: 0-500ms (display as ms, store as seconds for synth.set calls)
- decay: 0-1000ms (display as ms)
- sustain: 0-1 (step 0.01)
- release: 0-3000ms (display as ms)
- cutoff: 500-8000 Hz
- vibrato depth: 0-0.3 (step 0.01, strings only)

**Update logic — on slider change:**
Get instruments from `getSynthInstruments()`. For each instrument in the map, check if it has the expected update method (use `'updateEnvelope' in instrument` type guard). Call the matching method:
- ADSR slider change → `(instrument as any).updateEnvelope({ attack: val })` (or decay/sustain/release)
- Filter cutoff → `(instrument as any).updateFilterCutoff(hz)`
- Vibrato depth → `(instrument as any).updateVibratoDepth(depth)`

Since instruments are keyed by program number in loadedInstruments, and we want to update only the relevant category, use the programRange to filter which instruments to update. Map the program range to the correct key(s) in the loadedInstruments map.

**Display sliders using native HTML input[type="range"]** — no external slider library needed. Style with Tailwind: `w-full accent-cyan-400`.

Labels show current values in human-readable units: "Attack: 5ms", "Cutoff: 3800Hz".

**Important:** Use `useCallback` or direct inline handlers for slider onChange — do NOT debounce, changes must be instant for real-time feedback.
