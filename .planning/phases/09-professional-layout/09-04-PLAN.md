   - Color: cyan-400 or similar accent color
   - Triangle size: ~10px wide, ~8px tall
   - Cursor: grab on hover, grabbing during drag

3. Drag behavior:
   - onMouseDown on triangle starts drag
   - window.addEventListener('mousemove') during drag
   - window.addEventListener('mouseup') ends drag
   - Pass delta to parent via onDrag callback
   - Clean up listeners in useEffect return

4. Positioning:
   - Absolute positioned at `left: {position}px`
   - Parent (TimelineRuler) provides position based on playhead ticks

Implementation pattern (similar to ResizeHandle):
```tsx
const handleMouseDown = (e: React.MouseEvent) => {
  e.preventDefault();
  const startX = e.clientX;

  const handleMouseMove = (moveEvent: MouseEvent) => {
    const deltaX = moveEvent.clientX - startX;
    onDrag(deltaX);
  };

  const handleMouseUp = () => {
    window.removeEventListener('mousemove', handleMouseMove);
    window.removeEventListener('mouseup', handleMouseUp);
    onDragEnd();
  };

  window.addEventListener('mousemove', handleMouseMove);
  window.addEventListener('mouseup', handleMouseUp);
};
```
