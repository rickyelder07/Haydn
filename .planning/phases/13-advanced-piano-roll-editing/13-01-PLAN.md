**QuantizeParams interface:**
```typescript
export interface QuantizeParams {
  gridTicks: number;      // e.g. ppq/4 for 1/16 note
  strength: number;       // 0–100 (percentage)
  thresholdPct: number;   // 0–100 (skip notes already within this % of grid)
  swingPct: number;       // 50–75 (50 = no swing)
}
```

**quantizeNote function:** Takes `originalTicks: number` and `params: QuantizeParams`, returns new tick position. Algorithm:
1. Find nearest grid line: `gridPosition = Math.round(originalTicks / gridTicks)`, `nearestGrid = gridPosition * gridTicks`
2. Determine if offbeat: `isOnOddGrid = gridPosition % 2 !== 0`
3. Apply swing: `swingOffset = isOnOddGrid ? ((swingPct - 50) / 50) * gridTicks : 0`, `swungGrid = nearestGrid + swingOffset`
4. Threshold guard: `distanceToGrid = Math.abs(originalTicks - swungGrid)`, `threshold = (thresholdPct / 100) * gridTicks`. If `distanceToGrid < threshold` return `originalTicks` unchanged.
5. Strength interpolation: `return Math.round(originalTicks + (strength / 100) * (swungGrid - originalTicks))`

**gridTicksForValue function:** Takes `ppq: number` and `value: string`, returns tick count for that note value:
- `'1/4'` → `ppq`
- `'1/8'` → `ppq / 2`
- `'1/16'` → `ppq / 4`
- `'1/32'` → `ppq / 8`
- `'1/4T'` → `Math.round(ppq * 2 / 3)`
- `'1/8T'` → `Math.round(ppq / 3)`
- `'1/16T'` → `Math.round(ppq / 6)`
- Default: `ppq / 4`

Use `Math.round()` on all triplet values to avoid floating-point accumulation. Export both functions and the interface.

**DetectedChord interface:**
```typescript
export interface DetectedChord {
  bar: number;             // 0-indexed bar number
  barStartTicks: number;
  barEndTicks: number;
  symbol: string;          // e.g. "Cmaj", "Dmin", "G7"
  noteIndices: number[];   // indices into the notes array for this bar
}
```

**detectChordsPerBar function:** Takes `notes: HaydnNote[]`, `ppq: number`, `timeSignature: { numerator: number; denominator: number }`. Returns `DetectedChord[]`.

Algorithm:
1. `ticksPerBeat = ppq * (4 / timeSignature.denominator)`, `ticksPerBar = ticksPerBeat * timeSignature.numerator`
2. Guard: if `ticksPerBar <= 0 || notes.length === 0` return `[]`
3. `maxTick = Math.max(...notes.map(n => n.ticks + n.durationTicks))`
4. `numBars = Math.ceil(maxTick / ticksPerBar)`
5. For each bar 0..numBars-1:
   - Collect note indices where `note.ticks >= barStart && note.ticks < barEnd`
   - Skip if fewer than 3 notes
   - Get unique pitch classes: `Array.from(new Set(barNoteIndices.map(i => Note.pitchClass(Note.fromMidi(notes[i].midi)))))`
   - Run `Chord.detect(pitchClasses)` — if empty array returned, skip
   - Prefer non-slash chord: `const preferred = detected.find(c => !c.includes('/')) ?? detected[0]`
   - Format via `formatChordSymbol(preferred)` — if null returned, skip
   - Push `{ bar, barStartTicks: barStart, barEndTicks: barEnd, symbol, noteIndices: barNoteIndices }`

**formatChordSymbol function:** Takes `chordName: string`, returns `string | null`.
- Return null if `chordName.includes('/')`
- Call `Chord.get(chordName)` — if `!chord.tonic` return null
- Map `chord.type` to shorthand suffix using this map:
  - `'major'` → `'maj'`, `'minor'` → `'min'`, `'augmented'` → `'aug'`, `'diminished'` → `'dim'`
  - `'dominant seventh'` → `'7'`, `'major seventh'` → `'maj7'`, `'minor seventh'` → `'min7'`
  - `'diminished seventh'` → `'dim7'`, `'half-diminished'` → `'m7b5'`
  - `'suspended fourth'` → `'sus4'`, `'suspended second'` → `'sus2'`
  - Fallback: use `chord.type` as-is
- Return `chord.tonic + suffix`

Import: `import { Chord, Note } from 'tonal';` and `import type { HaydnNote } from '@/lib/midi/types';`

Export all three: `DetectedChord`, `detectChordsPerBar`, `formatChordSymbol`.

updateNoteDirectly: (noteIndex, updates) => {
  const state = get();
  if (state.selectedTrackIndex === null) return;

  const project = useProjectStore.getState().project;
  if (!project) return;

  const currentNotes = [...project.tracks[state.selectedTrackIndex].notes];
  if (noteIndex < 0 || noteIndex >= currentNotes.length) return;

  currentNotes[noteIndex] = { ...currentNotes[noteIndex], ...updates };

  // Write directly to project store — do NOT call state._history.push()
  syncToProjectStore(state.selectedTrackIndex, currentNotes);
  // Note: canUndo/canRedo flags intentionally unchanged
},
