Create `src/components/PianoRoll/ChordStrip.tsx` as a `'use client'` component. This is a DOM-based component (not canvas) — chord labels are absolutely positioned divs.

**Props interface (export as ChordStripProps):**
```typescript
export interface ChordStripProps {
  chords: DetectedChord[];       // From chordDetection.ts
  scrollX: number;
  zoomX: number;
  ppq: number;
  width: number;                 // Total canvas width (same as piano roll)
  onChordClick: (chord: DetectedChord) => void;
}
```

**Layout:** A fixed-height strip (24px). The strip has left padding of `PIANO_KEY_WIDTH` (60px) to align with the piano roll note grid (not over the piano keys).

```tsx
<div
  className="relative overflow-hidden bg-[#0D1117] border-b border-white/8"
  style={{ height: '24px', width: `${width + PIANO_KEY_WIDTH}px` }}
>
  {chords.map((chord, i) => {
    const pixelsPerTick = 0.1 * zoomX;
    const x = PIANO_KEY_WIDTH + ticksToX(chord.barStartTicks, pixelsPerTick, scrollX);
    const barWidth = (chord.barEndTicks - chord.barStartTicks) * pixelsPerTick;

    // Skip if outside visible area
    if (x + barWidth < 0 || x > width + PIANO_KEY_WIDTH) return null;

    return (
      <button
        key={i}
        onClick={() => onChordClick(chord)}
        className="absolute top-0 h-full flex items-center px-1.5 text-[10px] font-mono text-cyan-400/70 hover:text-cyan-400 hover:bg-cyan-500/10 rounded transition-colors truncate"
        style={{
          left: `${x}px`,
          maxWidth: `${Math.max(20, barWidth - 2)}px`,
        }}
        title={`${chord.symbol} — click to select notes`}
      >
        {chord.symbol}
      </button>
    );
  })}
</div>
```

Import `DetectedChord` from `./chordDetection`. Import `PIANO_KEY_WIDTH`, `ticksToX` from `./gridUtils`.

The component is purely presentational — no internal state. It receives pre-computed `chords` from the parent (PianoRollEditor will use `useMemo` to compute them).
