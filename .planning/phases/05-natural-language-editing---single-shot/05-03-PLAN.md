         - onClick: call store.setPrompt(localInput) then store.submitEdit()

           - Show: error message + dismiss button (X)
           - bg-red-900/30 border border-red-800/50 text-red-300
           - onClick dismiss: store.dismissError()
           - Persists until dismissed
         - Submitting state (status === 'submitting' or 'executing'): gray info bar
